## 项目介绍

使用GitHub Actions定时检查禁漫APK是否有更新，如果有，就下载一份保存到Release中。

[✨✨✨点我去APK下载界面✨✨✨](https://github.com/hect0x7/JMComic-APK/releases/latest)
